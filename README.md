# Gesture Controlled Snake Game

A modern twist on the classic Nokia Snake Game — controlled entirely using **hand gestures** via webcam.

No keyboard. No mouse. Just your hand 

---

## Features
- Real-time hand gesture control
- Index finger movement to control direction
- Built using Computer Vision & AI
- Smooth and touch-free gameplay
- Nostalgic snake game experience

---

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Pygame
- NumPy

---

## How It Works
- Webcam tracks your hand using MediaPipe
- Index finger movement determines direction:
  - Move right → Snake goes right
  - Move left → Snake goes left
  - Move up → Snake goes up
  - Move down → Snake goes down
